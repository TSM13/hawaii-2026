# Hawaii 2026 Trip Planner 🌺

A custom travel-planning website for our October 2026 Hawaii trip:
**Los Angeles → Big Island → Kauai → Maui | Oct. 1–17, 2026**

## View the site

👉 **Live site:** https://tsm13.github.io/hawaii-2026/

## What's inside

- Full daily itinerary with expandable details for each day
- Filter days by island (Big Island / Kauai / Maui)
- Recommended booking links for the must-do experiences
- A trip planning checklist
- Mobile-friendly layout and a print-friendly view

## Editing

The entire site is a single file: [`index.html`](index.html). Open it in any
browser to preview, edit the HTML, then commit and push — GitHub Pages updates
the live site automatically within a minute or two.
