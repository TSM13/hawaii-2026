# Hawaii 2026 Trip Planner 🌺

A custom travel-planning website for our October 2026 Hawaii trip:
**Los Angeles → Big Island → Kauai → Maui | Oct. 1–17, 2026**

## View the site

👉 **Live site:** https://tsm13.github.io/hawaii-2026/

## Pages

- **[`index.html`](index.html)** — home: countdown, island heroes, where we're staying (with
  maps + drive times), daily itinerary, booking links, saving checklist.
- **[`calendar.html`](calendar.html)** — October 2026 month grid, color-coded by island; each
  day links to its itinerary details.
- **[`dining.html`](dining.html)** — curated kid-friendly restaurants near each stay, Kauai
  beachfront luau picks, and an editable reservations tracker (saved per-device).
- **[`littleones.html`](littleones.html)** — tips for traveling with the toddlers + infant.
- **[`styles.css`](styles.css)** — shared Hawaiian/Polynesian theme used by every page.

## Editing

Edit the HTML/CSS, then commit and push — GitHub Pages updates the live site automatically
within a minute or two. Images live in [`images/`](images/); day cards have `id="day-oct-N"`
anchors that the calendar links to.
